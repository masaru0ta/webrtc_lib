# 機能一覧

## カプセル戦記テスト用ゲーム

|  #  | カテゴリ | 機能名 | 概要 |
|-----|---------|--------|-----|
| 1-1 | 戦略パート | マップ表示 | 24x24グリッドのマップを表示、地形タイル描画 |
| 1-2 | 戦略パート | ユニット配置 | ユニットをマップ上に配置・表示 |
| 1-3 | 戦略パート | ユニット移動 | ユニット選択、移動範囲表示、移動実行 |
| 1-4 | 戦略パート | ターン制 | 青軍・赤軍の交互ターン管理 |
| 2-1 | 戦略パート | 占領システム | 都市・基地の占領機能 |
| 2-2 | 戦略パート | 生産システム | CPを消費してユニット生産 |
| 3-1 | 戦闘パート | アクションバトル | 横スクロール格闘戦闘 |
| 4-1 | 対戦機能 | WebRTC接続 | P2Pでの2人対戦 |
